# Библиотека - Spring BOOT REST API
1. Создано Spring Boot приложение.
2. Реализован REST API для сущности Book (тема "Библиотека").
3. Реализованы:
 - GET endpoint с @RequestParam для поиска по любому полю Book
 - GET endpoint с @PathVariable для id
4. Реализованы слои: Controller → Service → Repository.
5. Реализованы DTO и mapper между Entity и API-ответом.
6. Настроен Checkstyle, приведён код к Google style.
